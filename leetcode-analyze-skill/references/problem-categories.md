### 4.2 同类型题目推荐规则

当分析某道题目时，推荐相似题目的策略：

1. **相同算法模式**：使用相同的解题技巧
2. **相同数据结构**：操作相同的数据结构
3. **难度递进**：从Easy到Medium到Hard
4. **知识点关联**：涉及相同的核心概念

**示例**：分析"两数之和"后推荐
- LeetCode 15 - 三数之和（扩展到三个数）
- LeetCode 167 - 两数之和 II（有序数组，双指针优化）
- LeetCode 170 - 两数之和 III（数据结构设计）

## 6. 使用建议

1. **系统学习**：按照数据结构或算法类型，从易到难逐步攻克
2. **举一反三**：完成一道题后，立即练习同类型的2-3道题
3. **定期复习**：使用间隔重复法，定期回顾已完成的题目
4. **总结模式**：每完成一个专题，总结该类型题目的通用模板
5. **模拟面试**：定期从高频题中随机抽取题目进行限时练习
